# Install
```console
brew install gpatch
brew install opam
opam install dune 
opam install co http.lwt graphql-lwt yojson
```
# Build
```console
dune build main.exe
```
# Run
```console
dune exec main.exe
```

# Graphql URL
http://localhost:8080/graphql
